# 이동희 · Frontend Developer

주어진 범위의 구현에 머무르지 않고, 프로젝트의 맥락을 이해하며 필요한 문제를 해결합니다. 개발 조직이 없는 환경에서 퍼블리셔로 웹 개발을 시작해, 에이전시 프로젝트 수행부터 사내 프로젝트의 설계·개발·운영, 웹 개발 프로세스 구축까지 업무 범위를 확장해왔습니다.

프로젝트 전반을 혼자 책임지고 완성해온 경험을 바탕으로, 개발 조직 안에서 설계와 기술적 판단을 논의하고 검증하며 팀의 성과에 기여하고자 합니다.
